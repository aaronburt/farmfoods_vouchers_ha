# Privacy Policy

We value your privacy. This integration is built with privacy and local control at its core.

### 1. No Telemetry or Tracking
This integration does not contain any tracking, telemetry, analytics, or user profiling code. 

### 2. Local Storage
All configuration details (your voucher page URL) and state tracking data (which vouchers have been marked as used) are stored strictly locally on your Home Assistant server. No data is sent to the maintainers or any third-party analytics platforms.

### 3. Outbound Requests
The integration only makes outbound HTTP requests to the official Farmfoods server (`https://www.farmfoods.co.uk`) to scrape your personal voucher page. No other network requests are made.
